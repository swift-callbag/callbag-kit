**Examples**

```swift
  let source = from(0...5)

  _ = source
    |> last()
    |> forEach(print) // 5
```

```swift
  let source = from(0...5)

  _ = source
    |> last(3)
    |> forEach(print) // 3
                      // 4
                      // 5
```
